API Basejump: Image Search Abstraction Layer

User stories:

1) I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
2) I can paginate through the responses by adding, ?offset=2 , parameter to the end of the URL.
3) I can get a list of the most recently submitted search strings.

Example creation usage:

https://imaginary-bank.glitch.me/search/[your search term]

https://imaginary-bank.glitch.me/search/[your search term]?offset=[a number to paginate]

Example search usage:

https://imaginary-bank.glitch.me/search/presidents?offset=10

Example output:

{ "image_title": "some random image", "image_url": https://www.somewebsite.com/randomimage.jpg" }

View search history:

https://imaginary-bank.glitch.me/history
